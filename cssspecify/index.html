อะไรคือ css specify !?

CSS specify คือ

มันคือการวัดลำดับความสำคัญของคำสั่งในตัวไฟล์ CSS ของเราที่ใช้คู่กับตัว HTML นั่นเอง

ซึ่งความสำคัญที่ว่านี้จะมองในรูปแบบของตัวเลขยิ่งเลขเยอะ

ความสำคัญก็ยิ่งมากพอความสำคัญเยอะกว่า คนอื่นก็จะได้รับการเอาใจใส่กว่าคนอื่นๆ

จะมองเรื่องนี้ในรูปแบบของพลัง ใครพลังมากกว่าก็ชนะไป !!

SPECIFY ต่างจาก CASCADE อย่างไร

- SPECIFY จะมองที่คนที่พลังสูงสุดก่อน

- CASCADE เมื่อเทียบกับ CASCADE แล้วจะมองลำดับแทน

EXAMPLE

เราจะมาทำการแสดงผลตรงบรรทัดที่ 8 ว่าผลเป็นอย่างไรโดยการสั่งที่ CSS

ผลออกมาก็คือ !!

ถึงเราจะกำหนดให้เป็น GRAY ก่อนที่ h1 แต่ค่าพลังก็สู้ .eiei h1 ไม่ได้

วิธีการนับค่าพลัง

คงจะพอเห็นตัวอย่างบ้างแล้วต่อไปก็จะมาดูวิธีการนับลำดับหรือค่าพลังนั่นเอง

ถ้าเขียนแบบ style = สีเลย + 1000

ถ้าใส่ ID ค่าพลัง + 100

ถ้าใส่ CLASS ค่าพลัง + 10

ใส่ ELEMENT HTML ค่าพลัง +1
